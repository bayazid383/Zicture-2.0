# Zicture_The_Online_Shopping

📸 Zicture 2.0
Zicture 2.0 is a dynamic web application built using PHP and MySQL. It provides features like product browsing, cart management, user authentication, and admin control, designed with a clean and scalable structure.
________________________________________
🚀 Features
•	🔐 User Login & Authentication

•	🛒 Shopping Cart System

•	📦 Product Management

•	❤️ Wishlist & Compare System

•	📊 Admin Panel

•	🗄️ MySQL Database Integration

•	🎨 Responsive UI (CSS + JavaScript)
________________________________________
🛠️ Technologies Used
•	Frontend: HTML, CSS, JavaScript

•	Backend: PHP

•	Database: MySQL

•	Server: XAMPP

•	Dependency Management: Composer

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/bayazid383/Zicture-2.0.git
2️⃣ Move project to XAMPP
C:\xampp\htdocs\
3️⃣ Start server
•	Open XAMPP

•	Start Apache and MySQL
________________________________________
4️⃣ Setup Database
1.	Open phpMyAdmin
2.	Create a new database (e.g., zicture_db)
3.	Import your .sql file
________________________________________
5️⃣ Run the project
http://localhost/Zicture
